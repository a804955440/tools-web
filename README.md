## XTools

![xtools logo](/public/logo.png)

XTools is a free, open-source, and data-secure toolbox.


## Feature

### Image
- Compress
- Convert
- Removebg
- Mosaic

### Video

- Compress

### Audio
- Compress

### Date
- dayjs utils
- momentjs utils
- date utils

### Text

- Converter

- Diff

### Writer
- word-count
